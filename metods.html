while 

                     
   Оператор while в JavaScript позволяет воспроизводить цикл, который выполняет задачу столько раз, пока истинно заданное условие¹. Цикл while имеет следующий синтаксис:
           
    while (условие) {
      // код
    }
                 
   Код из тела цикла выполняется, пока условие истинно⁴. Предикат - это условие, которое указывается в скобках после while. Это условие вычисляется и проверяется перед выполнением тела цикла на каждой итерации³.
                      
   Вот пример использования цикла while в JavaScript:
                
    let i = 0;
    while (i < 3) { // выводит i, пока i < 3
      alert( i );
      i++;
    }
                   
   В результате выполнения этого кода будут выведены числа от 0 до 2⁴.

   Конечно! Вот еще несколько примеров использования цикла while в JavaScript:

  let i = 0;
  while (i < 10) {
    console.log(i);
    i++;
  }
  
  В результате выполнения этого кода будут выведены числа от 0 до 9.
  
  
  let i = 0;
  while (i < 10) {
    if (i === 5) {
      break;
    }
    console.log(i);
    i++;
  }
  
  В результате выполнения этого кода будут выведены числа от 0 до 4.
  
  
  let i = 0;
  while (i < 10) {
    if (i === 5) {
      i++;
      continue;
    }
    console.log(i);
    i++;
  }
  
  В результате выполнения этого кода будут выведены числа от 0 до 9, кроме числа 5.

Оператор расширения (`...`) 

                     
    Оператор расширения (`...`) в JavaScript используется для передачи элементов массива или объекта в качестве аргументов функции или для создания нового массива или объекта на основе существующих.

    Пример использования оператора расширения (`...`) для передачи элементов массива в функцию:
                   
      const arr = [1, 2, 3];
      console.log(...arr); // Вывод: 1 2 3
                 
    В этом примере оператор расширения (`...`) используется для передачи элементов массива `arr` в функцию `console.log()`. Функция выводит каждый элемент массива в консоль.
                      
    Пример использования оператора расширения (`...`) для создания нового массива на основе существующих:
                 
      const arr1 = [1, 2, 3];
      const arr2 = [4, 5, 6];
      const arr3 = [...arr1, ...arr2];
      console.log(arr3); // Вывод: [1, 2, 3, 4, 5, 6]
                      
    В этом примере оператор расширения (`...`) используется для создания нового массива `arr3` на основе существующих массивов `arr1` и `arr2`. Массив `arr3` содержит все элементы из массивов `arr1` и `arr2`.
    
    Вот еще несколько примеров использования оператора расширения (`...`) в JavaScript:

    1. Копирование массива:

    const arr1 = [1, 2, 3];
    const arr2 = [...arr1];
    console.log(arr2); // Вывод: [1, 2, 3]
    
    2. Объединение массивов:
    const arr1 = [1, 2, 3];
    const arr2 = [4, 5, 6];
    const arr3 = [...arr1, ...arr2];
    console.log(arr3); // Вывод: [1, 2, 3, 4, 5, 6]
    
    3. Передача аргументов функции:
    function sum(a, b, c) {
      return a + b + c;
    }
    const arr = [1, 2, 3];
    console.log(sum(...arr)); // Вывод: 6

    4. Создание нового объекта на основе существующего:
    const obj1 = { a: 1, b: 2 };
    const obj2 = { c: 3, d: 4 };
    const obj3 = { ...obj1, ...obj2 };
    console.log(obj3); // Вывод: { a: 1, b: 2, c: 3, d: 4 }

String.prototype.repeat()

Метод repeat() конструирует и возвращает новую строку, содержащую указанное количество соединённых вместе копий строки, на которой он был вызван. 
                     
Синтаксис
str.repeat(count)

Примеры
'абв'.repeat(-1);   // RangeError
'абв'.repeat(0);    // ''
'абв'.repeat(1);    // 'абв'
'абв'.repeat(2);    // 'абвабв'
'абв'.repeat(3.5);  // 'абвабвабв' (количество будет преобразовано в целое число)
'абв'.repeat(1/0);  // RangeError

({ toString: () => 'абв', repeat: String.prototype.repeat }).repeat(2);
// 'абвабв' (метод repeat() является обобщённым методом)

Array.prototype.reduce() 

                     
  Метод reduce() применяет функцию reducer к каждому элементу массива 
  (слева-направо), возвращая одно результирующее значение.                
  Метод .reduce() может использоваться для сложения элементов массива:
  const numbers = [1, 2, 3, 4, 5];
  const sum = numbers.reduce((acc, cur) => acc + cur);
  console.log(sum); // 15
  
  
  Также можно использовать цикл for:
  const numbers = [1, 2, 3, 4, 5];
  let sum = 0;
  for (let i = 0; i < numbers.length; i++) {
    sum += numbers[i];
  }
  console.log(sum); // 15

Array.prototype.every() 

                     
  Функция every() - это метод, доступный в JavaScript, 
  который принимает массив и функцию в качестве аргументов. 
  Он проверяет, являются ли все элементы массива истинными 
  в соответствии с заданной функцией. Если все элементы 
  массива удовлетворяют условиям функции, то функция 
  возвращает true. Если хотя бы один элемент не проходит 
  проверку, то возвращается значение false.
  
  Например, если мы хотим проверить каждый элемент массива, 
  чтобы убедиться, что все числа являются положительными, 
  мы можем использовать метод every() следующим образом:
  
  const числа = [4, 5, 2, 8, 10];
  const все_положительные = числа.every(function(элемент) {
    return элемент > 0;
  });
  console.log(все_положительные); // true

  В данном примере наша функция-аргумент проверяет, что 
  каждый элемент массива больше 0. Так как все элементы 
  массива удовлетворяют этому условию, метод every() 
  возвращает true.
  
  Более формально, every() принимает функцию, которая 
  имеет до трех аргументов (element, index и array) в 
  качестве параметров и возвращает логическое 
  значение (true / false). Если функция возвращает 
  false для одного из отдельных элементов, every() 
  возвращает false. Если функция вернула true для
   каждого элемента, every() возвращает true.

Array.prototype.map()

Метод `map()` в JavaScript создаёт новый массив с результатом вызова указанной функции для каждого элемента массива¹. Синтаксис метода: 
        
   const new_array = arr.map(function callback(currentValue[, index[, array]]) {
     // Возвращает элемент для new_array
   }[, thisArg])
- `callback` - функция, которая будет вызвана для каждого элемента массива.
- `currentValue` - текущий обрабатываемый элемент массива.
- `index` (необязательный) - индекс текущего обрабатываемого элемента в массиве.
- `array` (необязательный) - массив, который вызывает метод map().
- `thisArg` (необязательный) - объект, который будет использоваться как `this` при вызове функции callback. 
                     
     Пример использования метода map() на объекте строки String 
     для получения массива байт в кодировке ASCII, представляющего значения символов:
     const map = Array.prototype.map;
     const charCodes = map.call('Hello World', (x) => x.charCodeAt(0));
     // теперь charCodes равен [72, 101, 108, 108, 111, 32, 87, 111, 114, 108, 100]
     1. Пример использования метода `map()` для переворачивания строки:
     const string = '12345';
     const reversed = Array.prototype.map.call(string, (x) => x).reverse().join('');
     // reversed равен '54321'
     2. Пример использования метода `map()`
      для итерации элементов массива:
     const sweetArray = [2, 3, 4, 5, 35];
     const sweeterArray = sweetArray.map(sweetItem => {
       return sweetItem * 2;
     });
     console.log(sweeterArray);
     // Вывод: [4, 6, 8, 10, 70]
     3. Пример использования метода `map()` для создания 
     нового массива из объектов:
     const arr = [
       { name: 'apple', count: 2 },
       { name: 'orange', count: 5 },
       { name: 'pear', count: 3 },
     ];
     const newArr = arr.map(item => {
       return item.name;
     });
     console.log(newArr);
     // Вывод: ['apple', 'orange', 'pear']
     4. Пример использования метода `map()` для создания 
     нового массива из объектов с изменением свойств:
     const arr = [
       { name: 'apple', count: 2 },
       { name: 'orange', count: 5 },
       { name: 'pear', count: 3 },
     ];
     const newArr = arr.map(item => {
       return {
         label: item.name,
         value: item.count * 2,
       };
     });
     console.log(newArr);
     // Вывод:
     // [
     //   { label: 'apple', value: 4 },
     //   { label: 'orange', value: 10 },
     //   { label: 'pear', value: 6 }
     // ]
     5. Пример использования метода `map()` для создания нового
      массива из объектов с изменением свойств и фильтрацией:
     const arr = [
       { name: 'apple', count: 2 },
       { name: 'orange', count: 5 },
       { name: 'pear', count: 3 },
     ];
     const newArr = arr.map(item => {
       if (item.count > 2) {
         return {
           label: item.name,
           value: item.count * 2,
         };
       }
     });
     console.log(newArr);
     // Вывод:
     // [
     //   undefined,
     //   { label: 'orange', value: 10 },
     //   undefined
     // ]

Array.prototype.filter() 

                     
    Метод filter() - это итеративный метод, который создает новый массив, заполненный элементами, которые проходят тест, предоставленный функцией. Он не выполняет функцию для пустых элементов и не изменяет исходный массив¹²³.

    Вот пример использования:
                   
    const words = ['spray', 'limit', 'elite', 'exuberant', 'destruction', 'present'];
    const result = words.filter(word => word.length > 6);
    console.log(result);
    // ожидаемый вывод: Array ["exuberant", "destruction", "present"]
                   
    В этом примере мы создаем массив слов и затем используем `filter()`, чтобы создать новый массив со всеми словами, которые имеют более шести букв¹.

    Вот еще несколько примеров:

    const numbers = [1, 2, 3, 4, 5];
    const evenNumbers = numbers.filter(number => number % 2 === 0);
    console.log(evenNumbers);
    // ожидаемый вывод: Array [2, 4]
    
    const fruits = ['apple', 'banana', 'grape', 'orange'];
    const longFruits = fruits.filter(fruit => fruit.length > 5);
    console.log(longFruits);
    // ожидаемый вывод: Array ["banana", "orange"]

    В первом примере мы создаем массив чисел и затем используем filter(), чтобы создать новый массив только с четными числами.

    Во втором примере мы создаем массив фруктов и затем используем filter(), чтобы создать новый массив только с фруктами, длина которых больше пяти букв.

Array.prototype.includes() 

                     
 Метод includes() определяет, содержит ли массив определённый элемент, 
 возвращая в зависимости от этого true или false.
 Синтаксис
 arr.includes(searchElement[fromIndex = 0])

 includes() - это метод JavaScript, который используется для определения,
 содержится ли определенная строка в другой строке.
 Например, если у вас есть строка "hello world", вы можете использовать 
 includes() для определения, содержит ли она слово "world".
 Пример использования:
 const str = 'hello world';
 console.log(str.includes('world')); // true
 console.log(str.includes('foo')); // false
 
 Этот метод возвращает значение true, если переданная подстрока найдена, 
 в противном случае возвращает значение false.

 const array1 = [1, 2, 3];

console.log(array1.includes(2));
// Expected output: true

const pets = ['cat', 'dog', 'bat'];

console.log(pets.includes('cat'));
// Expected output: true

console.log(pets.includes('at'));
// Expected output: false

Array.prototype.slice() 

                     
Метод slice() возвращает новый массив, содержащий копию части исходного массива.
Синтаксис
arr.slice([begin[, end]])

const animals = ['ant', 'bison', 'camel', 'duck', 'elephant'];

// Expected output: Array ["camel", "duck", "elephant"]

console.log(animals.slice(2, 4));

// Expected output: Array ["ant", "bison", "camel", "duck", "elephant"]

for() 

                     
 Оператор for создаёт цикл, состоящий из трех необязательных выражений,
 заключенных в круглые скобки и разделенных точкой с запятой, за 
 которыми следует инструкция (часто блок инструкций), которая должна 
 выполняться в цикле.
 Синтаксис
 for ([инициализация]; [условие]; [финальное выражение])
   [инструкция]

let str = '';
for (let i = 0; i < 9; i++) {
  str = str + i;
}

console.log(str);
// Expected output: "012345678"

new Set() 

                     
   new Set()- это конструктор в JavaScript, который создает 
   новый объект Set.Объект Set - это коллекция уникальных 
   значений, которые могут быть любого типа: и чисел, 
   и строк, и объектов.
     
   Каждое значение может быть добавлено в Set только один раз, 
   что гарантирует отсутствие дубликатов. Также Set обеспечивает 
   быстрый доступ к значениям и имеет ряд удобных методов для 
   работы с ними.
   
   Для создания объекта Set используется конструктор `new Set()`, 
   который не принимает аргументов. Можно передать ему 
   итерируемый объект, например массив, для инициализации 
   начальных значений:
   
   const mySet = new Set([1, 2, 3]);
   console.log(mySet); // Set { 1, 2, 3 }
     
   Также можно добавлять и удалять значения из Set с помощью 
   методов `add()` и `delete()`:
   
   mySet.add(4);
   mySet.delete(2);
   console.log(mySet); // Set { 1, 3, 4 }
                      
   В целом, Set позволяет эффективно решать множество типичных 
   задач, связанных с уникальными значениями, поиском пересечений, объединения и т.д.
